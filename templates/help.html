How do I add a new product?

Admins can add products via the Add Product link in the dashboard or navigation bar.

How do I restock inventory?

Admins can restock products using the Restock Product page.

How are expiry and low stock alerts shown?

Products near expiry or low in stock are highlighted in the inventory table with colored labels.

How do I back up or restore the database?

Use the db_backup.py script:
python db_backup.py backup to back up,
python db_backup.py restore <backup_file> to restore.

How do I access the REST API?

Login via the web UI, then access endpoints like /api/products for JSON data.

Who can manage users?

Only admin users can create, edit, or delete users.

Need more help?

Contact your system administrator or see the README for more details.
